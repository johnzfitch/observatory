# WebGPU Deepfake Detection - Project Context

## READ THIS FIRST

This is a WebGPU-based image classifier that detects AI-generated vs human-created images. 
It runs 4 ONNX models client-side in the browser.

## Current State (WORKING - DO NOT BREAK)

- InferenceEngine.js - maxConcurrency: 4, timeout: 30000
- 4 detection models: dima806_ai_real, smogy, umm_maybe, prithiv_v2
- ~400ms per model, ~1600ms total inference
- Output: confidence score (0-100% AI generated)
